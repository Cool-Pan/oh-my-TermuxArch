## 使用 Arch Linux

Arch Linux 只是众多 Linux 发行版中的其中一个，
与其它 Linux 发行版使用方式大同小异，这里不在赘述。

如果需要了解更多的有关于 Arch Linux 的信息，
请访问 Arch Linux 的官方 [Wiki]。

[Wiki](https://wiki.archlinux.org)
